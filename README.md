# 🩸 Non-Invasive Glucometer

> **An open-source hardware and software architecture for painless, real-time blood glucose monitoring.**

## 🌍 Project Overview
This repository contains the core technical development of a non-invasive glucometer prototype. The primary clinical objective of this engineering effort is to completely eliminate the need for traditional, invasive finger-prick blood sampling. 

By developing accessible, data-driven medical hardware, this project directly aligns with **GeoHealth** initiatives and **WHO** principles for improving global public health infrastructure.

## 🔬 Technical Architecture & Methodology
The system is built on a streamlined, low-latency hardware stack to ensure accurate biometric processing:

* **Hardware Platform:** Engineered natively on an **Arduino Uno** micro-controller ecosystem.
* **Sensing Technology:** Integrates specialized, safe infrared (IR) optical sensors.
* **Analytical Engine:** Evaluates and calculates blood glucose concentration utilizing real-time transmissive blood density analysis.
* **Core Language:** Programmed in **C** for optimized, hardware-level sensor processing and data extraction.
